This is and Application used NextJS, postgreSQL, socket.io

postgresSQL Version: 17.5.1

- Cài các package
  npm i

- Chạy Prisma để generate database
  npx prisma migrate
  npx prisma generate
  npx prisma db seed

- khởi động ứng dụng
  npm run dev

- build production
  npm run build
  npm run start
